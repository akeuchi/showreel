# Akeuchi Showreel

个人 Vibe Coding 作品集，展示前端页面设计与交互开发能力。

> 所有公司、品牌及产品名称均为虚构，与现实中的任何实体无关。

## 项目列表

| # | 名称 | 说明 |
|---|------|------|
| 01 | **LLM Atlas — 大语言模型全景图** | 探索主流大语言模型的能力边界与擅长方向 |
| 02 | **STUDIO FORM — 平面设计工作室** | 独立平面设计工作室作品集，品牌识别与编辑排版 |
| 03 | **AIANIME — AI 漫剧公司主页** | 简约科幻风格的 AI 漫剧创作公司官方主页 |

## 技术栈

- HTML / CSS / 原生 JavaScript
- Google Fonts（DM Sans、JetBrains Mono、Fraunces、Orbitron 等）
- 响应式设计，适配桌面与移动端

## 本地运行

```bash
# 克隆仓库
git clone <repo-url>
cd showreel

# 直接用浏览器打开
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

无需构建步骤，纯静态页面。
